第三方库的安装

第一种安装方法：pip（推荐）
cmd命令终端：pip -V  查看pip的版本
使用方法：pip install 第三方库的名字
   比如：pip install requests


第二种安装方法：
    pycharm直接安装：
    pycharm-->settings--->项目解释器-->  +


第三种安装方式：离线安装
    1、下载 XXX.tar.gz
       对下载压缩文件解压
       把解压后的文件copy到Python的安装路径下去
       cmd进入到 cd 进入到下载文件夹中
       执行命令：python setup.py install

    2、下载XXX.whl格式的文件

        直接cmd中执行 pip install  XXX.whl




PYPI：python的第三方库管理平台
